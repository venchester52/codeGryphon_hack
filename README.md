# AI-Analitics

Продуктовое Streamlit-приложение для быстрого маркетингового аудита и BI-аналитики по табличным данным.

Приложение:
- поддерживает загрузку **CSV, Excel, JSON, SQL и SQLite**;
- приводит источник к единому `pandas.DataFrame` перед анализом;
- автоматически сопоставляет колонки со внутренней схемой;
- динамически считает только доступные метрики;
- показывает слабые объявления и рекомендации;
- дает ответы AI-ассистента с учетом текущего контекста кампании;
- сохраняет историю анализов и чатов в SQLite;
- поддерживает регистрацию, вход и выход по email + пароль.

---

## Обновленный интерфейс и навигация

Интерфейс выполнен в более чистом и легком продуктовом стиле:

- Вверху расположен аккуратный **top header**.
- В левом верхнем углу находится кликабельный бренд **AI-Analitics**.
- Клик по бренду — это единое основное действие навигации:
  - возврат на главный экран (home/overview);
  - сброс временного drill-down состояния;
  - сохранение авторизации пользователя.
- Отдельная кнопка «Домой» в header отсутствует (дубликаты навигации убраны).
- Нативный sidebar Streamlit скрыт.
- Вместо него используется **правая панель управления**:
  - загрузка источника;
  - параметры источника;
  - открытие сохраненной сессии;
  - настройки дашбордов;
  - управление аккаунтом.
- Основная левая область остается рабочим пространством для:
  - обзора;
  - KPI;
  - метрик и результатов;
  - дашбордов;
  - рекомендаций;
  - AI-ассистента;
  - истории.

## Установка

```bash
python3 -m pip install -r requirements.txt
```

---

## Настройка `.env`

1. Скопируйте `.env.example` в `.env`.
2. Заполните переменные для GigaChat.
3. При необходимости настройте путь к SQLite.

Пример:

```env
GIGACHAT_CREDENTIALS=your_gigachat_authorization_key_here
GIGACHAT_SCOPE=GIGACHAT_API_PERS
GIGACHAT_CA_BUNDLE_FILE=/absolute/path/to/russian_trusted_root_ca_pem.crt

SQLITE_DB_PATH=data/marketing_auditor.sqlite3
DATABASE_URL=
```

## Поддерживаемые форматы

- CSV (`.csv`)
- Excel (`.xlsx`, `.xls`)
- JSON (`.json`)
- SQL (`.sql`)
- SQLite (`.sqlite`, `.sqlite3`, `.db`)

Для Excel/JSON/SQL/SQLite интерфейс показывает параметры выбора источника (лист, путь, таблица), если это нужно.

## Демо-сценарий

1. Запустите приложение.
2. Войдите или зарегистрируйтесь.
3. В правой панели загрузите файл (например, `sample_data/demo_campaign.csv`).
4. Проверьте вкладки:
   - `Обзор`
   - `Сопоставление данных`
   - `Метрики и результаты`
   - `BI-дэшборд`
   - `Рекомендации`
   - `AI-ассистент`
   - `История`
5. Нажмите на бренд **AI-Analitics** в левом верхнем углу — приложение вернет вас на главный экран.
6. Откройте сохраненную сессию через правую панель или вкладку истории и убедитесь, что контекст/чат восстановлены.

---

## Безопасность и хранение данных

- Аутентификация: email + пароль.
- Пароли хранятся как хэши Argon2 (`passlib`).
- История анализа и чаты сохраняются в SQLite через SQLAlchemy.
- Доступ к сессиям ограничен текущим авторизованным пользователем.
